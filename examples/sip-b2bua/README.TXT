SIP Back 2 Back User Agent Application
---------------------------------------------------
This is an b2bua application for the SIP11 RA. Its 
code should be taken as proper example.

Requirements: 
---------------------------------------------------
1. SIPP in OS path environment
2. Restcomm JAIN SLEE 2.x with SIP11 RA deployed
3. JBOSS_HOME pointing to Restcomm JAIN SLEE 2.x 

Running:
---------------------------------------------------
1. install sip11 ra
2. install this example
3. start Restcomm JAIN SLEE
4. cd to sipp, run first UAS then UAC, for either DIALOG (normal behavior) or CANCEL (...)
